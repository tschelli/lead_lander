## Key Features

1. **Progress Tracking** - Visual progress bar
2. **Conditional Logic** - Questions appear based on previous answers
3. **Point-Based Scoring** - Each answer adds points to different programs
4. **Smart Routing** - Guide users to appropriate programs
5. **Alternative Pathways** - Show GED/prerequisite programs when needed
6. **Mobile Optimized** - Full-screen modal on mobile
7. **Save Progress** - Can save and resume later (optional)
8. **Validation** - Real-time field validation
9. **Accessibility** - ARIA labels, keyboard navigation

This gives admins full control over the quiz flow while maintaining strong lead data!
